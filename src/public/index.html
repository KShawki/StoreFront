Advanced Full-Stack Web Development Nanodegree Program - EgFWD

About The Project

Build a JavaScript API based on a requirements given by the stakeholders. You will architect the database, tables, and columns to fulfill the requirements. Create a RESTful API to be accessible to the frontend developer. You will also have written test, secured user information with encryption, and provide tokens for integration into the frontend.

Users EndPoints:

  • Endpoint: /api/users/create
  • Method: POST

Query Parameters:

  • email: string
  • username: string
  • firstname: string
  • lastname: string
  • password: string

Example requests:

  • http://localhost:3000/api/users/create

  • Endpoint: /api/users/index
  • Method: GET

Example requests:

  • http://localhost:3000/api/users/index

  • Endpoint: /api/users/show/:id
  • Method: GET

Example requests:

  • http://localhost:3000/api/users/show/id

Product EndPoints:

  • Endpoint: /api/products/create
  • Method: POST

Query Parameters:

  • name: string
  • price: number
  • category: string

Example requests:

  • http://localhost:3000/api/products/create

  • Endpoint: /api/users/index
  • Method: GET
